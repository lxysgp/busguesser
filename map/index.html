How to play — Map Mode

Pick (or randomize) a Start and End bus stop. We’ll show both on the map, draw a line, and display straight-line distance. Use 📍 to set Start to the nearest stop to you. Share a challenge with 🔗 (it encodes stop codes in the URL).

  • R — Randomize
  • S — Swap Start/End
  • L — Locate nearest Start
  • / — Focus search
